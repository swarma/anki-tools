Markdown to Anki

把 Markdown 笔记转换为 Anki 卡片

使用说明

在左侧输入 Markdown 笔记, 在右侧获取 Anki CSV

示例输入（Markdown）

# 人际沟通入门
## 我们为什么要沟通？
### 生理需求
#### 有什么例子可以说明**沟通**在生理需求方面的作用？
##### 从越南战俘创造密码的例子可以看到，人际沟通对健康是不可或缺的
##### ![mark](http://pic.com/1.png)
### 认同需求
#### 我们对自我的认同，^^来源^^于哪里？
